  1. How to create new user credentials?
  2. 1. Open your Mac's Terminal app.
  3. 2. Type htpasswd -n -s NAME, and make sure to replace NAME with your Riskified username (your email address without '@riskified.com').
  4. 3. You'll be prompted to enter a new password, and repeat it - don't worry about the lock icon or about not seeing the characters you type, that's how it should be.
  5. 4. Send the text response (including the username) you receive to yoav@riskified.com. You'll be notified when your user credentials have been updated.
  1. How to decode my password hash?
  2. 1. Look for the hash that you sent Yoav by email. It should appear similar in format to lior.hemo:{SHA}cqhYQukOHF+omUkmDpDxu9b7xWY=.
  3. 2. Copy everything after the {SHA} substring (the hash itself), and try decoding it in one of the following sites. For that matter, any SHA-1 decoder can be used - no security issues.
    https://hashkiller.co.uk/sha1-decrypter.aspx http://md5decrypt.net/en/Sha1/ https://md5hashing.net/hash/sha1
  4. 3.Paste your hash and start praying to any/all gods. If you're lucky, the hash (which is a one-way function) will be decoded. Otherwise, please create new user credentials.
